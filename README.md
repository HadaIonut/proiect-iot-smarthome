# proiect-iot-smarthome

## Requirements:
 - python 3
   - FastAPI
   - uvicorn
- Arduino IDE

## Setup
 - Install Python
 - Install Arduino IDE
 - Enter pythionProject folder
 - run `pip install fastapi`
 - run `pip install uvicorn`
 - start the python server using `uvicorn main:app --reload --host 192.168.0.105`
 - upload the code to the arduino
